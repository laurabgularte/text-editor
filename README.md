# 📝 Text Editor (Python & Tkinter)

Um editor de texto desktop desenvolvido em Python. O aplicativo oferece uma interface gráfica completa para criação, abertura e edição de arquivos de texto, simulando as funcionalidades essenciais de editores como o Bloco de Notas.

## ✨ Funcionalidades

- **Manipulação de Arquivos**: Criar novo, abrir, salvar e "Salvar Como".
- **Edição Inteligente**: Suporte nativo para Desfazer (**Undo**) e Refazer (**Redo**).
- **Interface Fluida**:
  - Barra de rolagem vertical acoplada;
  - Quebra de linha automática;
  - Barra de status dinâmica que exibe o caminho do arquivo atual;
- **Alertas do Sistema**: Integração com caixas de mensagem nativas para confirmação de salvamento.

## 🛠️ Tecnologias

- **Python 3.x**
- **Tkinter**: Biblioteca padrão do Python para interfaces gráficas.

## 🚀 Como Executar o Projeto

Certifique-se de ter o Python instalado. Não é necessário instalar bibliotecas externas, pois o Tkinter já vem incluso no Python.

```bash
git clone [https://github.com/laurabgularte/text-editor.git](https://github.com/laurabgularte/text-editor.git)

```

```bash
cd text-editor
```

```bash
python text-editor-app.py
```
